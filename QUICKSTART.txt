🎯 WHAT HAPPENS WHEN YOU RUN IT
================================

1. Agent reads your product list (10 SKUs by default)

2. For each product, Claude AI:
   - Searches competitor websites
   - Finds matching products  
   - Extracts current prices
   - Compares vs your price
   - Generates recommendation

3. Results saved to pricing_data.db (SQLite database)

4. Text report saved to pricing_report.txt

5. View insights in dashboard (charts, alerts, opportunities)

Tips for best results:
• Use specific names: "Rx Clear 3-Inch Chlorine Tabs 50lbs"
• Include brand, size, quantity
• Match how competitors list products


📊 UNDERSTANDING OUTPUT
=======================

PRICE GAP %
-----------
-8%  = You're 8% cheaper than market (margin opportunity)
+12% = You're 12% more expensive (risk losing sales)  
0%   = Right at market average

RECOMMENDATIONS  
---------------
HOLD     - Current price is competitive
INCREASE - You're under-priced, capture margin
DECREASE - You're over-priced, adjust down

CONFIDENCE SCORE
----------------
85-100% = High confidence (exact product matches)
50-85%  = Medium confidence (similar products)
<50%    = Low confidence (verify manually)

🔧 TROUBLESHOOTING
==================

"No pricing data found" when opening dashboard
→ Run python pricing_agent.py first

"ANTHROPIC_API_KEY not found"  
→ Make sure you set the environment variable
→ Restart terminal after setting it

"JSON parsing error"
→ Check error message for details
→ Claude's response format might need adjustment

Competitor prices not found
→ Product name might be too vague
→ Try more specific search terms
→ Check products_sample.json for examples
